## 💬 QUICK Q&A

**Q: Where do I add properties?**
A: `src/data/properties.js` - See EDITING_GUIDE.md

**Q: How long does a change take?**
A: 5 min to change + 2-3 min to deploy = **7 min total**

**Q: What if something breaks?**
A: Developer fixes locally, tests, pushes. You monitor on Vercel dashboard.

**Q: Who gets GitHub access?**
A: Only your developer. Managers get Vercel dashboard access.

**Q: Where do I monitor?**
A: Vercel dashboard - https://vercel.com/dashboard
